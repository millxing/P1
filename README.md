# Brookline Precinct 1 Website

This repository contains a minimal static website for Brookline Precinct 1.

## GitHub Pages

Publish the site by pushing these files to a GitHub repository and enabling GitHub Pages from the repository root. The homepage must stay at `index.html` in the top level of the repository.

## File Map

- `index.html`: welcome page with quick access and precinct maps
- `tmms.html`: Town Meeting Members for Precinct 1
- `election-information.html`: polling place and election guidance
- `about.html`: placeholder precinct background page
- `warrant-may-2026.html`: placeholder warrant page
- `events.html`: placeholder events page
- `links.html`: civic and reference link directory
- `styles.css`: shared visual system
- `script.js`: shared navigation behavior
- `assets/`: source PDFs, images, and spreadsheet files used by the site

## Updating Content

- Edit page copy directly in the HTML files.
- Keep visual changes in `styles.css`.
- Keep JavaScript minimal in `script.js`.
- Do not publish private contact details from `assets/P1_TMMs.xlsx`; the public site only lists names and term expiration years.
